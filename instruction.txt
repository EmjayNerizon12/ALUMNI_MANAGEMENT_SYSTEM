Case Study Project: Alumni Survey System
Create a simple Alumni Survey Web System using Express and MySQL that includes the following features:

1. Alumni Account (Login Credentials)
Alumni must be able to register an account.
Alumni must log in using email/username + password.
Store credentials securely in the database.

2. Dynamic Questionnaire

Create a survey questionnaire for alumni.
Questions must be stored in the database (dynamic, not hardcoded).
Alumni answers must be stored per user.
At least 5 questions are required (e.g., employment status, skills, program relevance, etc.).

3. Job History Tracking
Alumni may add multiple job experiences.
Each job record must include:
Company name
Position
Start date
End date (or “Present”)

All job records must be stored in a separate table and linked to the alumni account.
